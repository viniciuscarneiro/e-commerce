# Front-end

## Description

Front-end using Nextjs

## Running

#### Must have installed

- Docker
- Docker Compose
- Node.js


#### Running MySQL database and RabbitMQ

```shell
docker compose up
```


#### Running the images server in another terminal:

```shell
npm install
npx serve -l 9000 images
```

#### Running the front-end app in another terminal:
```shell
npm run dev
```
